# 💼 Portfolio - Yurim

## 💡 Introduction

이 프로젝트는 웹 퍼블리셔 / 프론트엔드 개발자를 목표로 제작한 개인 포트폴리오 웹사이트입니다.  
디자인을 기반으로 HTML과 CSS를 활용하여 반응형 웹으로 구현하였습니다.

---

## 📌 About Project

본 프로젝트는 PC(1024px)와 모바일 환경을 고려하여 제작된 반응형 포트폴리오입니다.

Skills, About Me, Contact & Archive 섹션으로 구성되어 있으며  
카드형 UI를 활용하여 정보 전달력을 높였습니다.

레이아웃은 중앙 정렬을 기반으로 균형 있는 구조로 설계하였습니다.

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Media Query (Responsive Web Design)

---

## ✨ Features

- PC / Mobile 반응형 레이아웃 구현
- Skills & About Me 카드형 UI 구성
- 중앙 정렬 기반 레이아웃 설계
- Flex / Grid를 활용한 구조 배치
- GitHub / Contact 링크 연결

---

## 🎨 Design Highlights

- 1000px 기준 PC 레이아웃 설계
- 카드 UI 기반 정보 구조화
- 최소한의 컬러로 깔끔한 디자인 구성
- 균형 있는 좌우 대칭 레이아웃
- divider(구분선)를 활용한 섹션 분리

---

## ⚠️ Challenges & Solutions

- Flex와 Grid를 혼합 사용하면서 발생한 정렬 문제를 구조 재설계로 해결
- transform 기반 위치 보정 대신 wrapper 기준 정렬 방식으로 변경
- PC / Mobile 각각의 레이아웃을 Media Query로 분리하여 안정적인 반응형 구현
- 카드 정렬 및 중앙 기준선(divider) 정렬 문제 해결

---

## 📱 Responsive Design

- Mobile: 375px 기준 레이아웃
- PC: 1024px 기준 고정 레이아웃
- Media Query를 활용한 화면별 UI 최적화
